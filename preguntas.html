HTML

Introduction

What is HTML?

HTML stands for HyperText Markup Language. It creates web pages.

Explain the head and the body tags.

The <head> holds meta info, titles, links, and scripts. The <body> holds the visible content.

Basic Tags

Explain the section, article, aside, header, footer tags.

section: A block of related content.
article: Self-contained content like a post.
aside: Extra info or sidebar content.
header: The top part, often with a title or menu.
footer: The bottom part, for extra info.

CSS

Introduction

What is CSS?

CSS stands for Cascading Style Sheets. It styles HTML content.

What is the purpose of CSS?

It makes web pages look good by controlling design and layout.

How do you apply CSS to HTML? (3 options)

You can use inline styles, an internal <style> block, or an external CSS file.

What is a selector? What is a CSS rule?

A selector picks an HTML element to style. A CSS rule is the selector plus properties and values.

Basic CSS

Tag Selector: Styles all elements of a tag (e.g. p { ... }).

Class Selector: Styles elements with a specific class (e.g. .example { ... }).

Tag and Class Selector: Styles a tag with a specific class (e.g. p.example { ... }).

Id Selector: Styles an element with a unique id (e.g. #unique { ... }).

What is a class? What is an id? How are they related to CSS?

A class is a style you can reuse on many elements. An id is unique to one element. They help target elements with CSS.

Explain the CSS Box Model. Example:

The box model includes content, padding, border, and margin. For example:

div {
  padding: 10px;
  border: 1px solid black;
  margin: 5px;
}

How can you change the font color using CSS?

You can use the color property. For example: p { color: red; }
